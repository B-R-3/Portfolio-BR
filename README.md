# Portfolio de (futur) Développeur Web 💻🖥️

Bienvenue sur mon portfolio ! Ce projet est une vitrine de mes compétences et de mes projets réalisés en études et en autonomie. Il présente un design moderne avec une section dédiée à mes projets, organisés sous forme de grille interactive.

## Aperçu

Voici un aperçu visuel du portfolio avec une section d'accueil et une grille dynamique des projets réalisés :

- **Section d'accueil** : Avec un fond d'image en **parallaxe fixe**.
- **Grille de projets** : Responsive, interactif, chaque projet est cliquable avec des effets visuels.

## Fonctionnalités

- **Grille de Projets** : Affiche une grille de projets cliquables, adaptable à la taille de l'écran, avec des effets de survol.
- **Image de fond fixe** : L'image d'arrière-plan reste fixe lors du défilement, créant un effet de parallaxe.
- **Design réactif** : Compatible avec les appareils mobiles, tablettes et ordinateurs.
- **Lien vers LinkedIn** : Lien direct vers mon profil LinkedIn dans le pied de page.

## Installation

### 1. Prérequis
Assurez-vous d'avoir un navigateur moderne pour tester ce portfolio.

### 2. Cloner le projet
Clonez ce projet en local avec la commande suivante :

```bash
git clone https://github.com/B-R-3/portfolio.git
```

## Utilisation

1. Ouvrez le fichier `index.html` dans votre navigateur pour afficher le portfolio localement.
2. Naviguez à travers les différentes sections :
   - **Accueil** : Une image en arrière-plan fixe avec un effet de parallaxe.
   - **Projets** : Une grille de projets cliquables, chacun redirigeant vers des informations supplémentaires.
   - **Pied de page** : Contient un lien vers votre profil LinkedIn pour plus d'informations professionnelles.

## Technologies utilisées

- **HTML5** : Structure du contenu avec des balises sémantiques.
- **CSS3** : Utilisé pour le design, les effets de transition et la mise en page responsive.
- **JavaScript (ES6)** : Gère les interactions simples et les événements (exploitable dans `app.js` pour des fonctionnalités supplémentaires).
- **Google Fonts** : Intégration de la police *Agdasima* pour un design moderne et épuré.
